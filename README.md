# react-native-template

Template for `react-native` project by `datarockets`.

With preconfigured navigation and state management.

## Structure

### components
```text
src/components/AComponent
├── index.js
├── index.spec.js
└── styles.js
```

### screens
```text
src/components/AComponent
├── container.js
├── index.js
├── styles.js
└── view.js
```

### models
```text
src/models/AModel
├── actions.js
├── api.js
├── reducer.js
├── saga.js
├── selectors.js
└── types.js
```
